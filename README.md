# MSc-Artefact
Repository to store the developed artefact for MSc project

# Local Configuration

All files read in `.csv` format to maintain consistency accross all data pipelines. 

## Data folders:

1. `C:\IDR\RAW\Onc_ref\` - location of sharepoint list used to register new patients referred to Oncology.
2. `C:\IDR\RAW\ARIA_RT_Referral\` - location of report output from aria with new Radiotherapy referrals.
3. `C:\IDR\RAW\ARIA_CT\` - location of report utput from aria with Radiotherapy Planning CT appointments.
4. `C:\IDR\RAW\ARIA_Treat` - location of report utput from aria with Radiotherapy Treatment appointments.
